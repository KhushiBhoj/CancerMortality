### Strategic Analysis of Cancer Incidence and Mortality in the United States

**CancerMortality** is a data analytics project that leverages **American Cancer Society** data to study cancer incidence and mortality trends across the United States. The project combines statistical analysis, machine learning, and interactive visualization to generate actionable insights for targeted public health interventions.
